## 🚀 Overview

**DATAFLIX** is a cinematic, Netflix-style personal portfolio for **Ranadeep Dey**, an AI & Data Scientist with 3+ years of experience in Generative AI, Agentic Systems, LLMs, and Cloud Engineering.

The portfolio is structured around Netflix's iconic UI — a dramatic hero billboard, horizontally scrolling content rows for each portfolio section, and detailed modal popups that open just like Netflix show cards.

---

## ✨ Features

- 🎥 **Billboard Hero** — Full-screen cinematic header with name, title, synopsis, and CTA buttons (Play Resume, Download Resume, Episodes)
- 📜 **Scrollable Content Rows** — Each section (Experience, Skills, Achievements, Certifications, Education) is presented as a Netflix-style horizontally scrollable "row"
- 🪟 **Netflix-Style Modal Popups** — Click on any tile to open a detailed modal with description, metadata tags, and genre labels
- 🔍 **AI Chatbot Teaser** — A pill-shaped search bar in the navbar teases an upcoming Gemini-powered AI agent chatbot
- 🌙 **Cinematic Dark Theme** — Deep blacks, Netflix red accents, rich gradients, and smooth micro-animations
- 📱 **Fully Responsive** — Carefully optimized for Desktop, Tablet, and Mobile (including small phones ≤ 480px)
